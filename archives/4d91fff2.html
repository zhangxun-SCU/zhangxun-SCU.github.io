概率论基础知识

样本空间与随机事件

一个试验所有可能的结果成为样本空间，记为Ω\OmegaΩ，试验的每一个可能的结果是一个样本点，样本空间的一个子集称为随机事件

特殊的随机事件：

  • 必然事件：Ω\OmegaΩ,
  • 不可能事件：ϕ\phiϕ
  • 基本事件：由一个样本点组成的集合

事件的关系：

  • 包含、相等
  • 和\并
  • 积\交
  • 差：A-B = A-AB
  • 互斥（互不相容）：AB=ϕ\phiϕ
  • 互逆（对立）事件：A∪B=Ω且AB=ϕA\cup B = \Omega 且 AB=\phiA∪B=Ω且AB=ϕ
  • A−B=AB‾=A−ABA-B=A\overline{B}=A-ABA−B=AB=A−AB

完备事件组：A1,A2,...AnA_1, A_2, ...A_nA1​,A2​,...An​两两互斥，且A1∪A2∪...∪An=ΩA_1\cup A_2\cup ... \cup A_n=\OmegaA1​∪A2​∪...∪An​=Ω，则这n个事件为一个完备事件组。完备事件是可逆事件的延伸。

事件发生的概率

事件A再n次重复试验中出现k次,则k为事件A发生的频数，kn\frac{k}{n}nk​则为频率。

频率的性质:

  • 0≤fn(A)≤10\le f_n(A) \le 10≤fn​(A)≤1
  • fn(Ω)=1,fn(ϕ)=0f_n(\Omega)=1,f_n(\phi)=0fn​(Ω)=1,fn​(ϕ)=0
  • 由上一条，完备事件组中所有事件频率之和也为1

概率：在一个样本空间内，一个事件A，由P(A)P(A)P(A),且这个P函数满足非负性、规范性、可列可加性，则P(A)为A发生的概率

概率计算公式：

  • P(A−B)=P(AB‾=P(A)−P(AB))P(A-B) = P(A\overline{B} = P(A)-P(AB))P(A−B)=P(AB=P(A)−P(AB))
  • 加法公式:P(A∪B)=P(A)+P(B)−P(AB)P(A\cup B) = P(A) + P(B) - P(AB)P(A∪B)=P(A)+P(B)−P(AB)
  • 推广加法公式：(理解为多个由重叠的小图形面积去计算大图形面积)，P(∪i=1)=∑P(Ai)−∑P(AiAj)+∑P(AiAjAk)−...(−1)n−1P(A1A2...AnP(\cup _{i=1} ) = \sum{P(A_i) - \sum{P(A_i A_j) + \sum{P(A_i A_j A_k)- ...(-1)^{n-1}P(A_1 A_2 ... A_n} } }P(∪i=1​)=∑P(Ai​)−∑P(Ai​Aj​)+∑P(Ai​Aj​Ak​)−...(−1)n−1P(A1​A2​...An​

等可能概型

古典概型：有限个可能的结果 + 每种可能的结果出现的概率相等；计数、排列组合是计算古典概型的常用方法

几何概型：样本空间为无穷(即有无限个可能出现的结果)；将几何概型转化为图形长度、面积、体积占比是常用的求解方法

条件概率

条件概率：在事件B发生的条件下A发生的概率P(A∣B)P(A|B)P(A∣B)
计算公式：P(A∣B)=P(AB)P(B)P(A|B) = \frac{P(AB)}{P(B)}P(A∣B)=P(B)P(AB)​

常用公式：P(A‾∣B)=1−P(A∣B)P(\overline{A}|B) = 1 - P(A|B)P(A∣B)=1−P(A∣B)

乘法公式：P(AB)=P(B)P(A∣B)=P(A)P(B∣A)P(AB) = P(B)P(A|B) = P(A)P(B|A)P(AB)=P(B)P(A∣B)=P(A)P(B∣A)

推广乘法公式：P(A1A2...An)=P(A1)P(A2∣A1)P(A3∣A1A2)...P(An∣A1A2...An−1)P(A_1 A_2 ... A_n) = P(A_1)P(A_2|A_1)P(A_3|A_1A_2)...P(A_n|A_1A_2...A_{n-1})P(A1​A2​...An​)=P(A1​)P(A2​∣A1​)P(A3​∣A1​A2​)...P(An​∣A1​A2​...An−1​)（看作很多个条件概率的展开）

全概率公式：P(B)=∑P(Ai)P(B∣Ai)P(B) = \sum{P(A_i)P(B|A_i)}P(B)=∑P(Ai​)P(B∣Ai​)

贝叶斯公式：P(A_i|B) = \frac{P(A_i)P(B|A_i)}{P(B)}=\frac{P(A_i)P(B|A_i)}{\sum{P(A_i)P(B|A_i)（**证明见ppt第一张第四节例5）

事件的独立性和伯努利概型

事件A与事件B的发生没有关系，及A、B独立：若P(AB)=P(A)P(B)P(AB)=P(A)P(B)P(AB)=P(A)P(B)，则称事件A和B相互独立。独立是两事件发生无关，与互斥有很大区别。

  • 若A、B独立，则A‾\overline{A}A与B、A与B‾\overline{B}B、A‾与B‾\overline{A} 与 \overline{B}A与B都独立

  • 两两独立：任意两个事件都独立

  • 相互独立：所有时间都独立

    n个事件相互独立则必定两两独立，反之则不然，（实例见ppt第一章第五节例3）

在实际应用上,我们通常根据事件的实际背景去判断独立性，而不是定义

小概率事件：概率小于0.05

如果一个试验重复进行n次，每次只要两种可能的结果——AAA与A‾\overline{A}A，这就是伯努利概型，设每次试验A发生的概率为p，则n此试验A发生k次的概率为**Pn(k)=Cnkpk(1−p)n−kP_n(k) = C_n^kp^k(1-p)^{n-k}Pn​(k)=Cnk​pk(1−p)n−k。此概率也被称为二项概率**。(**由此可以推广到多项概率，概率计算公式类似，见ppt第一章第五节18页)
